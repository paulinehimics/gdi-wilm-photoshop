Why Photoshop?

  • Photoshop was originally created as a photo editing program, hence the name.
  • Can be used to:
    • Create original artwork
    • Manipulate images
    • Retouch photographs
  • It is expensive ($$$). However, it's an industry standard and is widely used, and thus hard to completely break away from.
  • Popular upstart - Sketch!

Raster vs. Vector

Raster:

  • Dot-based
  • Dots = pixels (sometimes)
  • Cannot resize without quality loss
  • GIF, JPG, PNG
  • Photoshop

Vector:

  • Math-based (!)
  • Uses geometry to create graphics (!)
  • Infinitely resize while maintaining quality
  • SVG
  • Illustrator

Setting Up a New File

  • Create a new file (File > New)
  • Image Resolution: spacing of pixels in an image
  • Image Size: physical dimensions of an image
  • Default file format is PSD
  • Other file types: TIFF, JPG, GIF, PNG
